前期准备

注册成为高德地图开发者

百度搜索高德开放平台，进入页面后注册账号，然后登陆。

然后进入控制台，点击应用管理，我的应用。

点击新建应用，应用名称和应用类型看着取和选。

然后点击应用的这个加号按钮，添加key。

选择服务平台为Web端。

这样我们就获得了项目的key

安装并引入amap组件

使用npm install vue-amap --save命令安装amap组件

vue-amap其实能够抛开高德原生SDK覆盖大多数场景，但对于部分定制化成都较高的场景，可能还是需要引入高德原生SDK来支持。
