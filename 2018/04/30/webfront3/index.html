2.5 跨站之魂——Javascript：

  • 获取HTML的内容：document.getElementById(‘private_msg’).innerHTML。document对象代表整个DOM，getElementById()函数可与获取指定id号的标签对象，这些对象都有一个属性innerHTML，表示标签对象内的HMTL数据内容。当不知道id号时可以使用document.getElementsByTagName(‘dic’)[2].innerHTML。getElementsByTagName()函数通过标签名查找数据，返回一个数组，下标从0开始。

    document。cookie可以获得cookies，但并不是所有的cookies

    window.location或location处可以获取URL地址中的数据。

    AJAX全称Asynchronous JavaScript And XML，即异步的JavaScript与XML

    异步和同步对应，异步可以理解为单独开一个线程，独立与浏览器主线去做自己的事，这样浏览器就不会等待（阻塞），这个异步在后台悄悄进行，所以利用AJAX的攻击显得很诡异，无声无息。

    AJAX本身是由JavaScript构成的，XML并不是必须的，XML在这里是想值数据传输格式是XML，比如AJAX发送出去的HTTP请求，响应回的数据是XML格式。然后JavaScript去解析这个XML DOM树得到相应节点的内容。其实响应回的数据可是还可以是JSON、文本、HTML等。

    AJAX的核心对象是XMLHttpRequest(一般简称xhr)，不过IE7之前的浏览器不支持xh对象，而是通过ActiveXObject来实现的（现在应该很少出现了）

    xhr实例化：


    var xmlhttp;
    if(window.XMLHttpRequest){
    xmlhttp = new XMLHttpRequest(); //IE7+,FireFox,Chrome,Safari等
    }else{
    xmlhttp = new ActiveXObject(“Microsoft.XMLHTTP”); //IE6,IE5
    }
    }
