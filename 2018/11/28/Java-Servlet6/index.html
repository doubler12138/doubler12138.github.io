JSTLеә“

1.дёӢиҪҪеҸҠдҪҝз”Ёпјҡ

дёӢиҪҪең°еқҖпјҡhttp://jstl.java.net

дҪҝз”Ёпјҡе°ҶдёӢиҪҪи§ЈеҺӢеҗҺзҡ„libзӣ®еҪ•дёӢзҡ„дёӨдёӘJARж–Үд»¶еӨҚеҲ¶еҲ°жҜҸдёӘеә”з”ЁзЁӢеәҸзҡ„WEB-INF/libзӣ®еҪ•дёӢ

2.JSTLж Үзӯҫеә“пјҡ

XML ж ёеҝғгҖҒжөҒжҺ§еҲ¶гҖҒиҪ¬жҚў http://java.sun.com/jsp/jstl/xml x
